This installer includes:

- **lewdware** — the main app
- **lewdware-config** — the configuration GUI
- **lw** — the CLI for building and testing modes

:::note[Windows SmartScreen]
On Windows, the installer is not code-signed. When prompted by SmartScreen, click **More info** →
**Run anyway** to proceed.
:::

:::note[macOS Gatekeeper]
The installer is not notarized. macOS will block it on first open.

After the first blocked attempt, open **System Settings → Privacy & Security**,
scroll to the security section at the bottom, and click **Open Anyway** next to the blocked
installer entry.
:::
