The Pack Editor is a graphical tool for creating and editing packs — import media, assign tags,
embed modes, and export for distribution.

:::note[macOS Gatekeeper]
The app is not notarized. After mounting the `.dmg` and dragging Pack Editor to Applications,
macOS will block it on first launch.

**Option 1 (quickest):** Right-click (or Control-click) the app in Applications and choose
**Open**, then click **Open** in the dialog.

**Option 2:** After the first blocked attempt, open **System Settings → Privacy & Security**,
scroll to the security section, and click **Open Anyway**.
:::
