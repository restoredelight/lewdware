title: Managing packs
description: Learn how to create a pack using the pack editor
---

Packs are just collections of images, videos and audio files. Creating and
editing packs is easy using the pack editor tool.

Begin by [downloading the pack editor for your OS](/download/pack-editor),
and installing it.

## Create a pack

Open the pack editor, and click _Create Pack_. Give the pack a name - you
can always change this later.

To upload media files, click _Import_, and then choose _Add files_ (to
add files individually) or _Add folder_ (to add a folder of files). Wait
for the files to be encoded and compressed (for long videos, this can take
a while).

Once you've uploaded your files, you can double click to view individual
files. Use Shift-Click or Ctrl-Click to select multiple files. Right Click
and press _Delete n items_ to delete the files you've selected.

At any point, press _Save_ to save the changes you've made.

Pressing the settings icon on the left allows you to edit the metadata
of the pack, which will be shown to a user when they select your pack.
