Lewdware is a pseudo-malware desktop app. When run, it spawns popups containing
images and videos, plays audio, opens links, changes your wallpaper, and
generally tries to annoy you as much as possible, and give you the sense of
your computer being taken over by a "virus". Primarily designed for
NSFW content, it was inspired by
[Edgeware++](https://github.com/araten10/EdgewarePlusPlus) and precursors.

Lewdware is highly customisable and scriptable, both through _packs_,
which contain images, videos and audio files, and _modes_, which change
the behaviour of Lewdware through Lua scripts.

Lewdware is not _actually_ a virus; it does not modify your personal files,
and the source code is [freely
available](https://github.com/restoredelight/lewdware).

## Get started

First, [download Lewdware for your OS](/download/), and use the installer
to install Lewdware. On Windows and macOS, you will likely get a warning about
an untrusted publisher - these are safe to bypass and ignore (I would have to pay
around $100/year to remove those warnings).

If at any point you want to uninstall Lewdware, it should be fairly simple to
do so. On Windows, just right click the program and click *Uninstall*. On
macOS, find the program in the Applications folder, and delete it.

Once you've got Lewdware installed, start it up. This should open up a
config window, allowing you to configure and run the main program. You will
need [a pack](/download/packs/).

Once you have downloaded a pack, go to *Packs and Modes* on the side menu,
click *Browse*, and choose the pack you downloaded. Then, go back to *x*, and
click *Launch* to start Lewdware.

## Stopping Lewdware

At any point, you can stop Lewdware, by pressing *Stop* in the config window,
by pressing a _panic key_ (which is <kbd>Ctrl</kbd> + <kbd>Esc</kbd> by
default), or by using the tray icon. Note that closing the config window will
_not_ stop Lewdware, allowing you to keep it running without having to keep the
config window open.

## Configuring lewdware

Lewdware provides a variety of modes by default, each of which have their own
set of options. Some packs also come with bundled modes, to provide a full
experience.
