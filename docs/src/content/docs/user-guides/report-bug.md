Found a bug in Lewdware, missing a feature or can't seem to get something to
work? You can [create an issue on GitHub](https://github.com/restoredelight/lewdware/issues/new/choose)
(note that you will need to create a GitHub account to do this).

## Sharing logs

The config app has an _Open logs folder_ button, which should take you to a
folder filled with logs. Sort the folder by time modified, newest first.

The prefixes of each folder correspond to the different apps of Lewdware:
`config-` contains logs for the config app used the launch Lewdware, `lewdware-`
contains the logs of Lewdware itself, and `pack-editor-` contains logs
for the pack editor. The suffixes correspond to the date of the logs
(YYYY-MM-DD). You probably want the suffix of the log file to be the
current date.

Once you have found the relevant log file, attach it to your issue.
